How the M&A integration works
Five-step pipelineRuns in minutes
01
Load target trial balance into BigQuery
Target provides a trial balance extract — SAP export, Excel, or CSV. Loaded into an isolated dataset: dise_target_[deal_name]. No access to acquirer data. Fully auditable.
dise_target_techco_acq · BigQuery dataset
02
Map target GL accounts to DISE categories
Claude reads target account descriptions and suggests DISE category assignments. Controller reviews and approves. Same methodology as acquirer — same ASC citations, same audit trail.
gl_dise_mapping · target_company_code
03
Run standalone pivot on target data
Same BKPF/BSEG/GL mapping three-way join — produces target's standalone DISE disclosure. Shows acquirer exactly what disclosure exposure they are acquiring.
dise_pivot · target · standalone
04
Run combined pro-forma pivot
UNION ALL of acquirer and target GL data. Single pivot produces the combined five-category table. Intercompany eliminations applied as adjusting entries. Category totals reconcile to combined income statement.
UNION ALL · pro_forma_combined · ASC 805
05
Generate combined footnote via Claude
Claude drafts the ASC 220-40 footnote for the combined entity including the pro-forma adjustment narrative required by ASC 805 for business combinations. Copy-paste ready for the first post-close 10-K or 10-Q.
ASC 220-40 · ASC 805 · Form 10-K
The BigQuery query pattern
-- Combined pro-forma DISE pivot (ASC 805) WITH combined_gl AS ( -- Acquirer actuals SELECT bseg.HKONT, bseg.DMBTR, 'ACQUIRER' AS entity FROM `[PROJ].CORTEX_SAP_CDC.bkpf` bkpf JOIN `[PROJ].CORTEX_SAP_CDC.bseg` bseg ON bkpf.MANDT=bseg.MANDT AND bkpf.BELNR=bseg.BELNR AND bkpf.GJAHR=bseg.GJAHR WHERE bkpf.GJAHR='2023' AND bkpf.BLART NOT IN ('AA','AF','AB') UNION ALL -- Target trial balance (loaded to BigQuery) SELECT gl_account AS HKONT, amount AS DMBTR, 'TARGET' AS entity FROM `[PROJ].dise_target_deal.trial_balance` WHERE fiscal_year='2023' ) SELECT m.expense_caption, ROUND(SUM(CASE WHEN LOWER(m.dise_category) IN ('inventory','purchases of inventory') THEN c.DMBTR ELSE 0 END),0) AS inventory, ROUND(SUM(CASE WHEN LOWER(m.dise_category) IN ('compensation','employee compensation') THEN c.DMBTR ELSE 0 END),0) AS compensation, ROUND(SUM(CASE WHEN LOWER(m.dise_category) IN ('depreciation','depreciation expense') THEN c.DMBTR ELSE 0 END),0) AS depreciation, ROUND(SUM(CASE WHEN LOWER(m.dise_category) IN ('amortization','intangible asset amortization') THEN c.DMBTR ELSE 0 END),0) AS amortization, ROUND(SUM(CASE WHEN LOWER(m.dise_category) IN ('other','other expenses') THEN c.DMBTR ELSE 0 END),0) AS other_exp, ROUND(SUM(c.DMBTR),0) AS total FROM combined_gl c JOIN `[PROJ].dise_reporting.gl_dise_mapping` m ON c.HKONT = m.gl_account WHERE m.status = 'mapped' GROUP BY 1 ORDER BY CASE m.expense_caption WHEN 'COGS' THEN 1 WHEN 'SG&A' THEN 2 WHEN 'R&D' THEN 3 ELSE 4 END
