Link 2 of 4 The DISE pivot query — how the number was calculated
This is the exact SQL that produced the $2,203,100 figure. It joins SAP journal entries from bkpf/bseg with your GL mapping table, filters to compensation accounts in SG&A, and sums the posted amounts. Run this query in BigQuery right now and it will return that exact number.
BigQuery SQL — DISE pivot (employee compensation / SG&A)
SELECT m.expense_caption, m.dise_category, ROUND(SUM(bseg.DMBTR), 0) AS total_amount, COUNT(*) AS posting_count, COUNT(DISTINCT bseg.HKONT) AS gl_accounts FROM `diplomatic75.CORTEX_SAP_CDC.bkpf` bkpf JOIN `diplomatic75.CORTEX_SAP_CDC.bseg` bseg ON bkpf.MANDT = bseg.MANDT AND bkpf.BUKRS = bseg.BUKRS AND bkpf.BELNR = bseg.BELNR AND bkpf.GJAHR = bseg.GJAHR JOIN `diplomatic75.dise_reporting.gl_dise_mapping` m ON bseg.HKONT = m.gl_account WHERE bkpf.GJAHR = '2023' AND m.status = 'mapped' AND m.dise_category = 'compensation' AND m.expense_caption = 'SG&A' AND bkpf.BLART NOT IN ('AA','AF','AB') GROUP BY 1, 2
