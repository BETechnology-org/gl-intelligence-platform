FILES IN THIS PACKAGE
─────────────────────

1. GL_Intelligence_Platform.html  ← START HERE
   The main platform — all modules in one app with sidebar navigation.
   Open in Chrome. Contains: Dashboard, GL Mapping, DISE Disclosure,
   Audit Trail, Anomaly Alerts, and Financial Q&A.

2. DISE_Compliance_Generator.html
   Standalone DISE disclosure generator with visual pill table.
   Use this for standalone demos or client presentations.

3. DISE_Anomaly_Alerts.html
   Tiered CFO alert system. Runs statistical anomaly detection
   against live BigQuery data. Generates CFO-quality alert emails.

4. DISE_Audit_Trail.html
   Interactive four-link audit chain demo. Works offline — no
   credentials required. Use for auditor and CFO demonstrations.

5. DISE_Financial_QA.html
   Natural language Q&A against live SAP data. Type a financial
   question, get BigQuery SQL and a plain-English answer.

6. DISE_Mapping_Toolkit.xlsx
   Excel workbook for GL account mapping — gap analysis, BigQuery
   field map, and disclosure preview across 5 tabs.

7. DISE_Expert_Guide.docx
   510-paragraph expert reference guide — solution architecture,
   demo script, stakeholder conversation guides, setup instructions,
   commercial model, and all SQL queries.

8. GL_Intelligence_Investor_Deck.pptx
   10-slide investor pitch deck — problem, solution, traction,
   market, roadmap, competition, business model, moat, and ask.
